# weather_forecast
A simple and efficient weather forecasting application that provides real-time weather updates and 7-day forecasts for any location worldwide.  ✨ Features 🌍 Search weather by city or coordinates  ⏱️ Real-time temperature, humidity, and wind speed  📅 7-day forecast with detailed conditions  🎨 Clean, modern UI for easy readability  ⚡ Fast API integration for accurate data
